Analyzer .NET Provider
======================

Current State: Proof of Concept ... Here be dragons.

This project implements a provider to enable code analysis of .NET, specifically
C#, projects for [analyzer-lsp](https://github.com/konveyor/analyzer-lsp)) 
using existing language server(s). Two providers that are publicly consumable
are:

 * [omnisharp-roslyn](https://github.com/OmniSharp/omnisharp-roslyn) - provides,
   [or at least it used to provide](https://github.com/dotnet/vscode-csharp#how-to-use-omnisharp),
   language services for [vscode-csharp plugin](https://github.com/dotnet/vscode-csharp)
   for Visual Studio Code
 * [csharp-language-server](https://github.com/razzmatazz/csharp-language-server)
   is a self-described hacky alternative to omnisharp-roslyn
